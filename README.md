# RouterOS CN IP List

CN ip list script generator for MikroTik RouterOS

## To use

```Ros Shell
/tool fetch https://raw.githubusercontent.com/RookieZoe/routeros-cnip-cidr/master/dist/cn_ip_cidr.rsc dst-path=cn.rsc;
/import file-name=cn.rsc;
```
